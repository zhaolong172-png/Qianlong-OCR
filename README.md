Qianlong OCR

Qianlong OCR is a browser-based OCR tool designed for PDFs, scanned documents, and images. Its goal is not just to extract text from images, but to generate searchable, copyable, and archivable OCR PDFs while preserving the original visual appearance of the document as much as possible. The entire processing pipeline runs locally in the browser without requiring server-side uploads, making it suitable for privacy-sensitive and security-conscious document workflows.

Built with Tesseract.js, PDF.js, pdf-lib, and FileSaver.js, the project supports PDF, PNG, JPG, and JPEG inputs, as well as drag-and-drop upload and multi-file processing. It also provides three OCR language options: Simplified Chinese + English, Traditional Chinese + English, and English only. To improve recognition quality across different document types, Qianlong OCR includes two optimization modes: Digital Screenshot Optimization and Automatic Segmentation for Scanned Documents.

The core idea behind Qianlong OCR is its dual-layer processing workflow. The project first renders the original page as the visual background, then creates a separate OCR-optimized canvas with grayscale conversion, contrast enhancement, and brightness adjustment for text recognition. After OCR is completed, the recognized text layer is merged back with the original full-color page background into a new PDF. This allows the output document to retain the look of the source file while also becoming searchable and selectable.

Qianlong OCR is well suited for scenarios such as invoice processing, academic paper archiving, contract search, and historical document digitization. For users who want a lightweight OCR solution that runs entirely locally and produces visually faithful searchable PDFs, it offers a practical and straightforward frontend-only approach.
